- On-Time Delivery Performance Monitoring

- Shipment Status Overview (Delivered, In Transit, Delayed)

- Fully Refreshable Dashboard

   USE supply_chain_db;
   SOURCE 01_schema.sql;
   SOURCE 02_sample_data.sql;
